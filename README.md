# 💰 Expense Tracker

A full-stack Expense Tracker application built using React.js (Vite) and Node.js.

🔗 GitHub Repo:  
https://github.com/taju089/Expenses-Tracker-Redux-Tansatck-Query

## 🚀 Features

- JWT Authentication
- Signup & Login
- Protected Routes
- Change Password
- Update Profile
- Create & Manage Categories
- Income & Expense Categories
- Create, Update & Delete Transactions
- TanStack Query for API Handling
- Redux Toolkit for User State Management

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Redux Toolkit
- TanStack Query
- React Router DOM
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
